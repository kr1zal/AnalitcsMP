Получить отчёт о стоимости размещения по товарам
post
/v1/report/placement/by-products/create

Описание и примерыКонсоль
Соответствует разделу FBO → Стоимость размещения в личном кабинете.

Отчёт можно получить не больше 5 раз в день.

header Parameters
Client-Id
required
string
Идентификатор клиента.

Api-Key
required
string
API-ключ.

Request Body schema: application/json
date_from
required
string
Дата начала отчётного периода в формате YYYY-MM-DD.

date_to
required
string
Дата окончания отчётного периода в формате YYYY-MM-DD.

Максимальный период — 31 день.
